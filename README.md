# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/gmorae/movi.it.git

# Go to server folder
$ cd movi.it

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```

Go to http://localhost:3000/ to see the result.
