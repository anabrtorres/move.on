# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/gmorae/move.on.git

# Go to server folder
$ cd move.on

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```

Go to http://localhost:3000/ to see the result.
